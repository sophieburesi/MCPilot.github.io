08 — Architecture
An architecture designed to never see your data
1
Collection — closest to the flow
Lightweight proxy or SDK wrapper intercepts each MCP call. Extracts metadata only: timestamp, agent ID, tool called, duration, return code.
SDK Python/TSKubernetes sidecarReverse proxy
2
Pipeline — real-time processing
Anonymization and enrichment before storage. Real-time anomaly detection via rules + lightweight ML model trained on MCP patterns.
OpenTelemetryKafka / streamingRules + ML
3
Storage — metadata only
Time-series for real-time metrics. Columnar storage for audit queries and long-term history. Configurable retention per client.
ClickHouseTimescaleDBEncryption at-rest
